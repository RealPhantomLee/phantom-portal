KEY FEATURES
================================================================================

SMART ROUTING:
  • Embedding requests → AI Pi's nomic-embed-text (5-10ms)
  • LLM requests → Main Pi (primary), AI Pi (fallback)
  • Automatic failover if primary unavailable
  • Can trigger model pull if needed

PERFORMANCE:
  • Embeddings: 5-10ms (local on AI Pi vs cloud latency)
  • Chat: 500-2000ms
  • Title generation: 300-800ms
  • Power: ~30-40W under load

RESILIENCE:
  • Health checks every initialization
  • Fallback routing if node unhealthy
  • Graceful degradation
  • Model auto-pull capability

================================================================================
TROUBLESHOOTING
================================================================================

Models not visible from Main Pi:
  1. Check network: ping 192.168.1.15
  2. Check Ollama running: curl http://192.168.1.15:11434/api/tags
  3. Check firewall: ufw allow 11434

Embedding requests slow:
  1. Verify nomic-embed-text on AI Pi
  2. Check network latency: ping 192.168.1.15
  3. Check AI Pi memory: ssh 192.168.1.15 "free -h"

Backend won't start:
  1. Check Python syntax: python -m py_compile backend/services/ollama_cluster.py
  2. Check imports: grep "from backend" backend/services/ollama_cluster.py
  3. Check logs: tail -f /var/log/phantom/backend.log

See AI_PI_SETUP.md for complete troubleshooting guide
