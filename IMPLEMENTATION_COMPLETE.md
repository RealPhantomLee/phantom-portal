### Your Setup Checklist (15 minutes)

1. **Generate TLS certificate**
   ```bash
   tailscale cert cyberdeck.REDACTED_TAILNET.ts.net
   ```

2. **Create and configure secrets**
   ```bash
   cp phantom.config.example.yaml phantom.config.yaml
   # Edit with: DeepSeek key, Telegram token, HA credentials, Blink password, VAPID keys
   ```

3. **Install dependencies**
   ```bash
   cd /home/jolly/Projects/phantom
   python3.14 -m venv venv && source venv/bin/activate
   pip install -r requirements.txt
   
   cd frontend
   npm install  # or pnpm install
   ```

4. **Run migrations**
   ```bash
   python -c "from backend.db.connection import run_migrations_sync; run_migrations_sync()"
   ```

5. **Start services** (choose one method)
   - **Systemd** (recommended): Copy service files to `/etc/systemd/system/` and start
   - **Docker**: `docker-compose up -d`
   - **Manual**: Run 3 terminals with backend, motion_monitor, frontend dev server

6. **Access portal**
   ```
   https://cyberdeck.REDACTED_TAILNET.ts.net
   ```
